## 5Ô∏è‚É£ Div vs Span (Very Important)

| Tag      | Purpose          | Display                |
| -------- | ---------------- | ---------------------- |
| `<div>`  | Block container  | Starts on **new line** |
| `<span>` | Inline container | Stays **in same line** |

```html
<div>Block content</div>
<span>Inline text</span>
```
